# Dashboard SOP & Regulasi BPS Kota Surabaya

Aplikasi web modern untuk manajemen dokumen Standar Operasional Prosedur (SOP) dan regulasi internal di BPS Kota Surabaya.

## 🚀 Tech Stack
- **Framework**: [React/Next.js] + Vite
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **UI Components**: shadcn/ui
- **Build Tool**: Vite
